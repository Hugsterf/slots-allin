# All In Mod

[![Thunderstore Download](https://img.shields.io/badge/Thunderstore-Download_Mod-3a0ca3?logo=thunderstore&logoColor=white)](https://thunderstore.io/c/gamble-with-your-friends/p/Hugsterf/AllInMod/)

Removes level-based MaxBet limits in casino games, allowing you to bet your entire balance.

## Download & Install

The easiest way to install is through the **Thunderstore Mod Manager** or **r2modman**:

1. Open your mod manager
2. Search for `AllInMod` by **Hugsterf**
3. Click **Download**

Or download manually from [Thunderstore](https://thunderstore.io/c/gamble-with-your-friends/p/Hugsterf/AllInMod/).

## Manual Installation

1. Install BepInEx 5.4.23+
2. Place `AllInMod.dll` into `BepInEx/plugins/AllInMod/`

## Features

- Bet any amount up to your current balance
- All button bets full balance
- Works on all games (slots, roulette, blackjack, etc.)
- Golden Chip uses original MaxBet (balanced)
