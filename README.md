# All-In Mod

Allows you to bet your entire balance in casino games, ignoring level-based MaxBet limits.

## Download
[Latest Release](https://github.com/Hugsterf/slots-allin/releases)

## Features
- Bet any amount up to your current balance
- All button bets full balance
- Works on slots, roulette, blackjack, and more

## Installation
1. Install BepInEx 5.4.23+
2. Place `SlotsAllInMod.dll` into `BepInEx/plugins/SlotsAllInMod/`
